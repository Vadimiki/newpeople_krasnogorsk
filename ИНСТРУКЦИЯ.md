# 🤖 Бот «Новые люди» — инструкция по запуску

## Что делает бот

1. Приветствует и кратко рассказывает о партии (со ссылкой на newpeople.ru)
2. Спрашивает возраст (18+) и гражданство РФ — если нет, вежливо завершает диалог
3. Даёт выбрать: **Сторонник** или **Член партии**
4. Присылает соответствующий файл анкеты для скачивания
5. Просит заполнить от руки, сфотографировать и прислать обратно в бот
6. Принимает фото/файл анкеты
7. Спрашивает ФИО и номер телефона для связи
8. Пересылает тебе как администратору: ФИО, телефон, Telegram, и саму заполненную анкету
9. В конце показывает рекламный блок про создание ботов на заказ — с кнопкой на твой Telegram

---

## Важно: персональные данные

Бот **не запрашивает паспортные данные внутри чата** — только ФИО и телефон для связи. Сами паспортные данные человек указывает в самой анкете (PDF/DOCX), которую заполняет от руки и присылает уже готовым файлом. Это снимает с бота функцию "оператора персональных данных" в чате — он просто пересылает файл тебе.

---

## Шаг 1 — Файлы анкет

В архиве уже лежат:
- `anketa_storonnik.pdf` — анкета сторонника
- `anketa_chlen.docx` — анкета члена партии

Они должны находиться **в одной папке с `bot.py`** — бот берёт их по этим именам.

---

## Шаг 2 — Создать бота в Telegram

1. Открой **@BotFather**
2. `/newbot` → задай имя, например `Новые люди — вступление`
3. Получи токен

## Шаг 3 — Узнать свой Chat ID

Напиши **@userinfobot** — он пришлёт твой числовой ID.

## Шаг 4 — Настроить bot.py

Замени в начале файла:
```python
BOT_TOKEN = "ВСТАВЬ_ТОКЕН_СЮДА"
ADMIN_CHAT_ID = "ВСТАВЬ_СВОЙ_CHAT_ID"
DEVELOPER_CONTACT = "@твой_username"   # твой контакт для рекламы услуг
```

## Шаг 5 — Запуск

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 bot.py
```

Должно появиться: `Бот «Новые люди» запущен ✅`

---

## Команды

- `/start` — начать заявку (для пользователей)
- `/cancel` — отменить (для пользователей)
- `/applications` — список последних заявок (только для администратора)

---

## Деплой на Railway (чтобы работал 24/7)

1. Залей `bot.py`, `requirements.txt`, `anketa_storonnik.pdf`, `anketa_chlen.docx` в репозиторий на GitHub
2. На railway.app → New Project → Deploy from GitHub Repo
3. В Variables добавь `BOT_TOKEN` и `ADMIN_CHAT_ID`
4. В Settings → Start Command впиши: `python3 bot.py`
5. Redeploy
